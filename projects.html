02
Télécoms / Data Engineering
Pipeline ETL Big Data - Suivi des Performances Commerciales

Développement d'un pipeline ETL distribué et scalable pour centraliser, nettoyer et analyser les flux transactionnels massifs et asynchrones d'un réseau de 50 agences télécoms (inspiré du modèle opérationnel d'Orange Cameroun). Le système ingère des formats hétérogènes, applique un alignement sémantique algorithmique et enrichit les données à grande échelle via des jointures de masse optimisées.

03
Médecine Prédictive / MLOps
Modèle IA de Prédiction du Diabète

Conception et déploiement d'un pipeline complet de Machine Learning pour le diagnostic précoce du diabète basé sur le dataset Pima Indians. Le projet englobe le nettoyage des données (gestion des zéros aberrants), l'entraînement d'un classifieur supervisé et l'exposition du modèle via une architecture découplée (API + UI web).

04
Télécoms / MLOps & Big Data
Système de prédiction du Churn

Développement d'un système prédictif de bout en bout visant à identifier les abonnés mobiles à risque de désabonnement. Ce projet intègre une phase d'ingénierie des caractéristiques haute performance avec Polars, l'optimisation d'un modèle Random Forest via RandomizedSearchCV (gestion des classes déséquilibrées), et l'encapsulation complète du pipeline de prétraitement et de scoring dans une API web industrielle.
